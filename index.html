Learn to Code

We are Hack Club. Here you can learn to build a responsive website and publish it, use common coding practices, learn the general process of coding, and much more. It is important that you click the sign up buttom so we can estimate how much to bring. Regular snacks include doughnuts, chips, and drinks!
